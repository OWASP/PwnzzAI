# Usage Policy

## Educational Purpose Only

PwnzzAI Shop contains **intentional security vulnerabilities**. It is designed
exclusively for:

- AI/LLM security education and training
- Hands-on workshop exercises
- Understanding how vulnerabilities work and how to defend against them

## Prohibited Uses

- Deploying in production environments
- Using the techniques demonstrated here against real systems
- Running the application on public-facing servers without proper network isolation
- Using the application to train LLMs or for any automated data collection
