# Roadmap

## Near-term Goals

- **Additional lab scenarios**: Expand OWASP Top 10 for LLMs coverage with
  more real-world exploit scenarios
- **Mitigation deep-dives**: Hardened-mode comparisons for each vulnerability
  (defensive coding patterns in Python, not just UI toggles)
- **Automated security scanning**: Scripts to verify lab exploit paths in CI

## Medium-term Goals

- **Self-guided learning paths**: Structured progression for beginners through
  advanced exploit techniques
- **CTF challenge expansion**: More challenge types for the CTFd workshop
  deployment
- **Multi-language support**: Localized lab descriptions and explanations

## Long-term Vision

- **AI Exchange alignment**: Continue mapping to the evolving OWASP AI Exchange
  threat taxonomy as it matures
- **Community lab contributions**: Framework for community-submitted labs with
  automated validation
- **Enterprise workshop kits**: Full facilitator kits with slide decks,
  handouts, and assessment rubrics
